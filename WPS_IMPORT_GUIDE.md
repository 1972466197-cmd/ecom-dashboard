# WPS 文档导入功能使用说明

## 📥 快速导入你的文档

### 方式一：直接导入页面

访问：**http://localhost:3000/wps/import**

1. 点击 **"使用示例"** 按钮（已预填你的文档链接）
2. 点击 **"🔍 分析文档"**
3. 选择要导入的工作表（总表/商品明细/直播数据）
4. 点击 **"✅ 确认导入"**

### 方式二：从 WPS 同步页面进入

1. 访问 http://localhost:3000/wps
2. 点击右上角 **"📥 导入文档"** 按钮

---

## 🎯 功能特性

### 支持的数据类型
- ✅ 抖音直播数据报表（销售额、订单数、转化率、GPM 等）
- ✅ 商品销售明细（SKU、价格、库存等）
- ✅ 订单汇总数据
- ✅ 库存预警表

### 导入流程
```
粘贴链接 → 分析文档 → 预览数据 → 选择工作表 → 确认导入 → 完成
```

### 数据预览
- 显示前 3 行数据预览
- 显示总行数
- 显示所有列名

### 导入历史
- 记录每次导入的文件名、工作表、行数、时间
- 显示导入状态（成功/失败）

## 📊 模拟导入数据示例

导入后会解析以下字段：

| 字段 | 说明 |
|------|------|
| 日期 | 销售日期 |
| 商品名称 | 商品标题 |
| SKU | 商品编码 |
| 销售额 | 当日销售金额 |
| 订单数 | 成交订单数量 |
| 转化率 | 观看 - 成交转化率 |
| GPM | 千次观看成交额 |
| 观看人数 | 直播间观看人数 |

## ⚠️ 注意事项

1. **当前为演示版本** - 实际解析 WPS 文档需要：
   - WPS Open API 授权
   - 或手动导出 Excel 后上传

2. **生产环境对接** - 需要：
   - 申请 WPS 开放平台 API
   - 配置 OAuth 授权
   - 实现真实的数据解析

3. **替代方案** - 如无法直接解析：
   - 从 WPS 导出 Excel 文件
   - 使用 Excel 上传功能导入
