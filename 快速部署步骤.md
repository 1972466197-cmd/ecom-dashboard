# 🚀 快速部署步骤（10 分钟上线）

## 方式一：Vercel 官网部署（最简单，推荐）

### 步骤 1：访问 Vercel
打开浏览器，访问：**https://vercel.com**

### 步骤 2：登录
- 点击 "**Log In**"
- 选择 "**Continue with GitHub**"（用 GitHub 账号登录）

### 步骤 3：导入项目
1. 点击 "**+ Add New Project**"
2. 选择 "**Import Git Repository**"
3. 选择你的 GitHub 账号
4. 找到 `ecom-dashboard` 仓库
5. 点击 "**Import**"

### 步骤 4：配置环境变量
点击 "**Environment Variables**"，添加：

```
名称：NEXT_PUBLIC_SUPABASE_URL
值：https://你的项目.supabase.co

名称：NEXT_PUBLIC_SUPABASE_ANON_KEY  
值：你的 anon public key
```

**如何获取 Supabase 密钥：**
1. 访问 https://app.supabase.com
2. 选择你的项目
3. 点击 Settings → API
4. 复制 Project URL 和 anon public key

### 步骤 5：点击 Deploy
点击 "**Deploy**" 按钮

### 步骤 6：完成！
等待 2-5 分钟，部署成功后获得网址：
```
https://your-project.vercel.app
```

## 方式二：GitHub + Vercel 自动部署

### 1. 推送到 GitHub

```bash
cd E:\山麓众创科技有限公司\ecom-dashboard

# 初始化 Git
git init

# 添加文件
git add .

# 提交
git commit -m "Initial commit"

# 创建 GitHub 仓库后，执行：
git remote add origin https://github.com/你的用户名/ecom-dashboard.git
git branch -M main
git push -u origin main
```

### 2. 在 Vercel 导入

1. 访问 https://vercel.com
2. 点击 "+ Add New Project"
3. 选择 GitHub 仓库
4. 点击 "Deploy"

### 3. 配置环境变量

在 Vercel 项目设置中添加：
- NEXT_PUBLIC_SUPABASE_URL
- NEXT_PUBLIC_SUPABASE_ANON_KEY

### 4. 完成！

以后每次推送到 GitHub 都会自动部署！
