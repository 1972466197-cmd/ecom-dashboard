# ⚙️ Vercel 环境变量配置

## 🚨 重要：必须配置环境变量！

部署失败是因为缺少 Supabase 环境变量。

---

## 配置步骤

### 第 1 步：打开 Vercel 项目

访问你的项目：
```
https://vercel.com/1972466197-cmds-projects/ecom-dashboard
```

### 第 2 步：进入设置

1. 点击项目顶部的 "**Settings**" 标签
2. 在左侧菜单选择 "**Environment Variables**"

### 第 3 步：添加环境变量

点击 "**Add New**" 按钮，添加以下变量：

#### 变量 1：
```
Key: NEXT_PUBLIC_SUPABASE_URL
Value: https://你的项目.supabase.co
Environment: Production, Preview, Development (全选)
```

#### 变量 2：
```
Key: NEXT_PUBLIC_SUPABASE_ANON_KEY
Value: eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...（你的 anon key）
Environment: Production, Preview, Development (全选)
```

### 第 4 步：保存

点击 "**Save**" 按钮

### 第 5 步：重新部署

1. 回到 "**Deployments**" 标签
2. 找到失败的部署
3. 点击 "**...**" → "**Redeploy**"

---

## 📋 如何获取 Supabase 密钥

### 1. 访问 Supabase
打开：https://app.supabase.com

### 2. 选择项目
选择你的项目

### 3. 进入 API 设置
点击左侧 "**Settings**" → "**API**"

### 4. 复制密钥

**Project URL:**
```
https://xxxxxxxxxxxxx.supabase.co
```

**anon public:**
```

```
