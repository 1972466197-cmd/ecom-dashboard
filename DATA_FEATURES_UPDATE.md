## 📝 使用说明

### 快速开始

1. **配置数据库**
   ```bash
   # 在 Supabase 创建项目
   # 运行 supabase-schema-full.sql 创建表结构
   ```

2. **配置环境变量**
   ```env
   NEXT_PUBLIC_SUPABASE_URL=https://your-project.supabase.co
   NEXT_PUBLIC_SUPABASE_ANON_KEY=your-anon-key
   ```

3. **启动开发服务器**
   ```bash
   npm install
   npm run dev
   ```

4. **导入数据**
   - 访问 `/import` 页面
   - 选择店铺和导入模板
   - 上传 Excel 文件
   - 点击"开始导入"

### 数据导入模板

所有模板都可以通过导入页面的"下载导入模板"按钮获取。

**必填字段：**
- 销售数据：日期、支付金额、支付订单数
- 店铺日概况：统计日期、店铺名称、支付金额
- 商品日概况：日期、商品名称、支付金额
- 刷单明细：主订单编号、商品标题、买家实付金额
- 退款明细：订单编号、退款编号、买家退款金额
