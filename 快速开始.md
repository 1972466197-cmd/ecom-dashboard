## 🎯 第一步：导入数据

### 方式 A：上传 Excel 文件（推荐批量导入）

1. 点击顶部菜单 **📥 数据导入**
2. 选择要导入的店铺（28 个店铺可选）
3. 上传 Excel 文件（支持 .xlsx, .xls）
4. 点击 **🚀 开始导入**

**Excel 模板格式（第一行表头）：**
```
日期，星期，支付金额，支付订单数，客单价，访客，加购，加购率，转化率，
全站推广，净利，...
```

**最少需要 3 列：**
- 日期（格式：2026-03-23）
- 支付金额
- 支付订单数

### 方式 B：手动编辑（适合少量数据）

1. 点击顶部菜单 **🏪 店铺销售情况**
2. 展开店铺分组（海林组/培君组/淑贞组/敏贞组）
3. 点击店铺名称展开每日数据
4. **双击任意单元格** 直接编辑
5. 修改后自动保存到数据库

## 💡 使用技巧

### 快速导入历史数据
1. 准备 Excel 文件，包含历史销售数据
2. 按店铺分别导入（每个店铺单独上传）
3. 系统会自动覆盖相同店铺 + 日期的数据

### 修改某一天的数据
- **单条修改**：在「店铺销售情况」页面双击单元格编辑
- **批量修改**：准备单行 Excel，重新导入

### 查看某个店铺的数据
1. 进入「店铺销售情况」
2. 选择对应分组
3. 点击店铺名称展开
4. 调整日期范围查看历史数据

## ⚠️ 注意事项

1. **日期格式**：必须为 `YYYY-MM-DD`（如 2026-03-23）
2. **数字格式**：不要带货币符号（¥、$），纯数字即可
3. **重复数据**：相同店铺 + 日期会自动覆盖
4. **空值处理**：空单元格默认为 0
