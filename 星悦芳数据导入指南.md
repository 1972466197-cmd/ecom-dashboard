## 🔧 步骤 1：在 Supabase 中创建数据表

### 1.1 打开 Supabase SQL Editor

1. 访问 https://app.supabase.com
2. 选择你的项目
3. 点击左侧 **SQL Editor**
4. 点击 **New Query**

### 1.2 执行 SQL 脚本

复制 `create-xingyuefang-tables.sql` 文件的全部内容，粘贴到 SQL Editor 中，然后点击 **Run**。

预期输出：
```
✅ 成功创建 4 个数据表
✅ 成功创建索引
✅ 成功创建 RLS 策略
```

## 🔍 常见问题

### Q1: 提示"table does not exist"
**解决：** 确保已在 Supabase 中执行了 `create-xingyuefang-tables.sql` 脚本

### Q2: 导入时提示"relation already exists"
**解决：** 表已存在，可以直接导入数据，无需重复创建表

### Q3: 数据导入后网站上看不到
**解决：** 
1. 检查是否已登录网站
2. 清除浏览器缓存
3. 重启开发服务器：`npm run dev`
